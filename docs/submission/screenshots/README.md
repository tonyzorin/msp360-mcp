# MCP App screenshot placeholders

Capture these in **Claude Desktop** after installing the extension with a sandbox tenant that has monitoring and RMM data.

Requirements ([submission guide](https://claude.com/docs/connectors/building/submission)):

- PNG format, at least **1000px** wide
- Crop to the **MCP App widget only** (no prompt text)
- 3–5 images for the directory carousel
- Provide matching prompt text in the submission form

| File | Tool | Prompt to use in Claude |
|------|------|-------------------------|
| `01-backup-monitoring.png` | `backup_rm_get_monitoring` | Show backup plan monitoring with status breakdown for all companies |
| `02-rmm-fleet.png` | `rmm_fleet_overview` | Show RMM fleet overview with online and offline endpoint counts |
| `03-backup-issues.png` | `backup_rm_list_issues` | List backup plan issues grouped by company from latest monitoring |

Replace this README with actual PNG files before submitting the desktop extension form.
